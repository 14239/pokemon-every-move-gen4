# 포켓몬 4세대 넓고 얕은 기술 챌린지

## 개요

어떤 포켓몬이든 존재하는 모든 기술을 쓸 수 있지만, 단 한 번씩만 쓸 수 있다고 할 때 엔딩을 볼 수 있을까요?

이 궁금증을 해결하려고 프로그램으로 만들어봤습니다.

원하는 기술을 선택하면 현재 대전중인 포켓몬이 그 기술을 시전합니다.

모든 기술 리스트, 사용한 기술 내역을 볼 수 있고, 진행 상태를 저장할 수 있습니다.

# 파일 구성

1. oneshot_allmove_script_dynamic.lua

    - BizHawk에 이 Lua 스크립트를 집어넣어야 합니다
    - 대전중인 포켓몬의 메모리 주소를 탐지하고, 기술을 바꿔주는 역할을 합니다

2. oneshot_allmove_challenge.exe

    - Python GUI 프로그램
    - 기술 선택 및 관리
    - 메모리 주소 자동 찾기 다이얼로그 포함

3. pokemon_moves.csv

    - 포켓몬 기술 데이터베이스 (467개 기술, 4세대 기준)

4. lua_interface/ (자동 생성 폴더)
    - command.txt: Python → Lua 명령 전송
    - result.txt: Lua → Python 스캔 결과 반환
    - memory_config.txt: 찾은 메모리 주소 저장

# 첫 사용 방법

1. BizHawk 에뮬레이터 실행 및 게임 로드 (설치링크 : https://github.com/TASEmulators/BizHawk/releases)

    - 포켓몬스터 PT기라티나, 하트골드/소울실버 지원 (DP는 테스트 안해봤습니다)

2. Lua 스크립트 실행

    - Tools → Lua Console
    - oneshot_allmove_script_dynamic.lua 열기
    - 콘솔에 "초기화 완료" 메시지 확인

3. 대전 시작

    - 게임에서 야생, 트레이너 상관없이 대전 시작

4. Python GUI 실행

    - python oneshot_allmove_challenge.exe
    - GUI 창 열림

5. 메모리 주소 자동 찾기
    - 메뉴: 도구 → 메모리 주소 자동 찾기
    - 현재 대전 중인 포켓몬이 보유한 기술 4개를 순서대로 선택
        - 기술 1, 2, 3, 4 드롭다운에서 선택
        - 빈 슬롯은 "0. (빈 슬롯)" 선택
        - 타이핑으로 검색 가능 (예: "불꽃", "1" 등)
    - "주소 찾기" 버튼 클릭
    - 스캔 완료 대기 (약 1~5초)
    - "메모리 주소를 찾았습니다!" 메시지 확인

# 이후 사용 방법 (메모리 주소 설정 완료 후)

1. 챌린지 저장/불러오기

    - 파일 → 새 챌린지: 새로운 챌린지 시작 (모든 기술 사용 기록 초기화)
    - 파일 → 챌린지 열기: 이전에 저장한 챌린지 불러오기 (.json 파일)
    - 파일 → 저장: 현재 챌린지 진행 상황 저장
    - 파일 → 다른 이름으로 저장: 새 파일로 저장
    - 저장 내용: 사용한 기술 목록, 사용 시간 기록 등
    - 저장 위치: saves/ 폴더에 .json 형식으로 저장됩니다

2. 기술 선택 및 사용

    - 사용할 기술 선택 (검색 가능)
    - "1P 기술 사용" 또는 "2P 기술 사용" 클릭
    - "2P 기술 사용"은 더블배틀의 오른쪽 포켓몬의 기술을 변경합니다
    - 게임에서 자동으로 기술 변경 및 터치 입력
    - 한번 사용한 기술은 해당 파일에서 다시 사용할 수 없습니다

3. 기술 관리 및 필터링
    - 검색: 기술명이나 ID로 빠른 검색
    - 타입 필터: 특정 타입 기술만 보기
    - 카테고리 필터: 물리/특수/변화 기술 구분
    - 사용 여부 필터: 사용한/안 한 기술만 보기

** 게임을 재시작하면 5단계(메모리 주소 자동 찾기)를 다시 해야 합니다! **

# 주의사항

1. 메모리 주소 스캔은 반드시 **대전 중**에 실행해야 합니다

    - 대전 시작 전/후에는 메모리 구조가 다릅니다

2. 선택한 기술이 정확해야 합니다

    - 게임 내 포켓몬의 기술 순서와 정확히 일치해야 합니다
    - 기술 선택란에서 좌측 상단 - 우측 상단 - 좌측 하단 - 우측 하단 순으로 기술을 적어주세요
    - 순서가 다르면 찾을 수 없습니다

3. 빈 슬롯 처리

    - 기술이 4개 미만이면 빈 슬롯은 "0. (빈 슬롯)" 선택

4. 게임 재시작 시

    - 메모리 주소가 변경되므로 다시 스캔해야 합니다
    - 같은 save state(강제 저장)를 계속 쓰면 주소가 동일할 수 있습니다

5. 1P/2P 주소
    - 일반적인 싱글 배틀에선 1P 기술 입력만 쓰시면 됩니다
    - 1P 주소를 찾으면 2P는 자동으로 +0x180 위치로 설정
    - 2P 기술은 더블 배틀 때 우측 포켓몬의 기술을 선택할 때 사용합니다

# 문제 해결

Q: "스캔 응답이 없습니다" 오류
A: Lua 스크립트가 실행 중인지 확인하세요

-   BizHawk Lua Console에서 스크립트 상태 확인
-   콘솔에 오류 메시지가 있는지 확인

Q: "일치하는 메모리 패턴을 찾을 수 없습니다" 오류
A: 다음을 확인하세요

1. 대전 중인지 확인
2. 선택한 기술이 실제 포켓몬 기술과 일치하는지 확인
3. 기술 순서가 정확한지 확인 (1번부터 4번까지)

Q: 기술 변경이 안 됩니다
A: 메모리 주소가 올바른지 확인

-   "Memory Config: Loaded" 상태인지 확인
-   게임을 재시작했다면 메모리 주소 재스캔 필요

Q: Python GUI에서 기술 목록이 안 보입니다
A: pokemon_moves.csv 파일이 같은 폴더에 있는지 확인
